Sammanfattning

QoplaSnap samlar inte in, lagrar eller delar någon personlig information. Tillägget är designat med din integritet i åtanke.

Vilken data samlas in?

Ingen personlig data samlas in. QoplaSnap:

  • Samlar inte in ditt namn, e-post eller annan personlig information
  • Spårar inte din webbhistorik
  • Använder inga cookies för spårning
  • Delar ingen data med tredje part

Hur fungerar tillägget?

När du använder QoplaSnap:

  • Läser tillägget URL:en från din aktiva flik (endast på qopla.com)
  • Skickar URL:en till vår server för att hämta produktbilder
  • Bilderna laddas ner direkt till din dator
  • All data raderas från servern efter nedladdning

Serverbehandling

Vår backend-server (hostas på Railway) behandlar tillfälligt restaurang-URL:er för att hämta bilder. Ingen data loggas eller sparas permanent. Sessionsdata raderas automatiskt efter att ZIP-filen laddats ner eller efter 30 minuter.

Dina rättigheter

Eftersom vi inte samlar in personlig data finns det ingen data att begära ut eller radera. Du kan när som helst avinstallera tillägget.

Ändringar i policyn

Om denna policy uppdateras kommer ändringarna att publiceras på denna sida med uppdaterat datum.
